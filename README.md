### 항상 도전하는 프론트엔드 개발자 김윤수입니다!

<br />
많은 서비스에서 데이터가 비즈니스 모델에 중요한 부분을 차지하는 시대라고 생각합니다.
<br />

<br />새로운 것을 두려워하지 않고 탐구적인 강점이 잘 드러날것이라고 생각하기 때문에 프론트엔드를 선택하게 되었습니다.
